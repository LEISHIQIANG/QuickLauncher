# 服务自启动测试指南

## 当前状态

✅ 代码已完成，包括：
- Windows 服务实现 (core/windows_service.py)
- 服务管理器 (core/service_manager.py)
- 安装包集成 (scripts/installer.iss)
- 设置界面集成 (ui/config_window/settings_panel.py)

## 测试步骤

### 1. 打包程序

```bash
# 使用你的打包脚本
python build.py
```

### 2. 测试安装包

1. 运行安装包
2. 勾选 "Start with Windows (Service Mode - Fastest)"
3. 完成安装
4. 检查服务：按 Win+R → 输入 `services.msc` → 查找 "QuickLauncher 快速启动服务"

### 3. 测试启动速度

1. 重启电脑
2. 记录从登录到托盘图标出现的时间
3. 预期：2-5 秒（比注册表方式快 10-20 秒）

### 4. 测试设置界面

1. 打开 QuickLauncher 设置
2. 取消勾选"开机自启"
3. 检查服务是否被卸载
4. 重新勾选，检查服务是否重新安装

## 验证清单

- [ ] 服务安装成功（services.msc 中可见）
- [ ] 服务状态为"自动"
- [ ] 重启后程序自动启动
- [ ] 启动速度 < 5 秒
- [ ] 设置界面可以启用/禁用
- [ ] 卸载程序时服务被清理

## 故障排查

**服务安装失败**
- 检查是否有管理员权限
- 查看日志：%APPDATA%\QuickLauncher\error.log

**服务已安装但程序未启动**
- 打开事件查看器 → Windows 日志 → 应用程序
- 查找 QuickLauncherService 相关错误

**启动速度仍然慢**
- 检查服务启动类型是否为"自动"
- 检查是否有杀毒软件拦截
