# 字体渲染优化说明

## 优化时间
2026-02-05

## 问题描述
设置窗口的字体看起来粗糙、不清晰，显示效果不佳。

## 优化内容

### 1. 全局字体渲染优化 (`main.py`)

**位置：** `main.py` 第335-350行

**改进内容：**
```python
# 优化字体渲染
default_font.setHintingPreference(QFont.HintingPreference.PreferFullHinting)
default_font.setStyleStrategy(QFont.StyleStrategy.PreferAntialias)  # 新增：抗锯齿
default_font.setWeight(QFont.Weight.Normal)  # 新增：正常字重
default_font.setKerning(True)  # 新增：启用字距调整
```

**说明：**
- `PreferFullHinting`: 使用完整的字体提示信息，使字形更清晰
- `PreferAntialias`: 优先使用抗锯齿渲染，使边缘平滑
- `Weight.Normal`: 设置标准字重（400），避免字体过粗
- `setKerning(True)`: 启用字距调整，使字符间距更自然

### 3. 字体大小标准化

**统一的字体大小规范：**

| 组件类型 | 字体大小 | 用途 |
|---------|---------|------|
| 标题 | 18-24px | 页面标题、对话框标题 |
| 副标题 | 14-16px | 章节标题、分组标题 |
| 正文 | 13px | 主要内容、标签、输入框 |
| 辅助文字 | 11-12px | 提示信息、状态栏、图标标签 |
| 工具提示 | 12px | 鼠标悬停提示 |

**配置窗口具体应用：**
- 文件夹列表：12px（紧凑显示）
- 图标名称：11px（小图标下方）
- 设置选项：13px（标准文本）
- 状态栏：11px（次要信息）
- 按钮：13px（操作文本）
- 输入框：13px（输入内容）

### ClearType 设置
Windows 的字体渲染质量还受到系统 ClearType 设置的影响。

**检查方法：**
1. 打开"控制面板" → "外观和个性化" → "字体"
2. 点击左侧"调整 ClearType 文本"
3. 按照向导调整以获得最佳显示效果

**推荐设置：**
- ✅ 启用 ClearType
- ✅ 使用子像素渲染
- ✅ 选择最清晰的文本示例

## Qt 字体渲染策略说明

### HintingPreference（字体提示）

| 策略 | 说明 | 效果 |
|-----|------|------|
| PreferDefaultHinting | 默认策略 | 由系统决定 |
| PreferNoHinting | 不使用提示 | 更平滑，但可能模糊 |
| PreferVerticalHinting | 仅垂直提示 | 折中方案 |
| **PreferFullHinting** | **完整提示（推荐）** | **最清晰，对齐像素网格** |

### StyleStrategy（样式策略）

| 策略 | 说明 | 效果 |
|-----|------|------|
| **PreferAntialias** | **优先抗锯齿（推荐）** | **边缘平滑** |
| NoAntialias | 不使用抗锯齿 | 边缘锯齿，但清晰 |
| PreferQuality | 优先质量 | 最佳渲染，较慢 |
| PreferMatch | 优先匹配 | 查找最佳字体匹配 |

## 思源黑体 vs 微软雅黑

### 思源黑体特点
- ✅ 字形现代、统一
- ✅ 中英文混排和谐
- ✅ 多字重支持（需额外字体文件）
- ✅ 开源免费
- ⚠️ 文件较大（10-15MB）

### 微软雅黑特点
- ✅ Windows 系统自带
- ✅ 渲染优化好
- ✅ ClearType 优化
- ⚠️ 字形相对传统

**当前实现：**
- 优先使用思源黑体（如果字体文件存在）
- 回退到系统思源黑体（如果已安装）
- 最终回退到微软雅黑

## 常见问题排查

### Q1: 字体仍然看起来粗糙
**可能原因：**
1. 字体文件未加载成功
2. 系统 ClearType 未启用
3. 显示器分辨率太低
4. DPI 缩放设置不当

**解决方法：**
1. 检查日志确认字体加载
2. 调整系统 ClearType 设置
3. 检查 Windows 缩放设置

### Q2: 不同窗口字体不一致
**可能原因：**
组件使用了内联样式覆盖了全局设置

**解决方法：**
搜索 `.setStyleSheet()` 调用，确保字体设置一致

### Q3: 字体文件缺失
**表现：**
日志显示 "字体文件不存在"

**解决方法：**
将 `SourceHanSansSC-Regular.otf` 放置到 `resources/fonts/` 目录

## 未来优化建议

### 1. 多字重支持
打包多个字重的思源黑体：

- `SourceHanSansSC-Regular.otf` (400)


### 2. 字体回退优化
```css
font-family: "Source Han Sans SC", "PingFang SC", "Microsoft YaHei",
             "Helvetica Neue", Arial, sans-serif;
```

### 3. 自适应字体大小
根据 DPI 缩放动态调整字体大小：
```python
def get_scaled_font_size(base_size: int) -> int:
    """根据DPI缩放获取字体大小"""
    screen = QApplication.primaryScreen()
    dpi = screen.logicalDotsPerInch()
    scale_factor = dpi / 96.0  # 96 DPI 是标准
    return int(base_size * scale_factor)
```

### 4. 用户自定义字体
在设置面板添加字体选择器：
- 允许用户选择系统字体
- 保存到配置文件
- 重启后应用
