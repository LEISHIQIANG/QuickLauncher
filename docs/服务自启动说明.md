# Windows 服务自启动方案

## 优势

- **启动最快**：开机即启动，无需等待桌面加载
- **优先级高**：系统服务级别，比注册表 Run 键早 10-20 秒
- **稳定可靠**：不受用户环境影响

## 使用方法

### 安装服务

1. 双击 `install_service.bat`
2. 在 UAC 提示中点击"是"授予管理员权限
3. 看到"服务安装成功"即可

### 卸载服务

1. 双击 `uninstall_service.bat`
2. 授予管理员权限
3. 服务将被移除

### 验证服务状态

按 `Win + R`，输入 `services.msc`，查找 "QuickLauncher 快速启动服务"

## 工作原理

1. 服务在系统启动时自动运行（后台）
2. 服务等待用户登录（检测 explorer.exe）
3. 用户登录后立即启动 QuickLauncher.exe
4. 主程序显示托盘图标和界面

## 注意事项

- 需要管理员权限安装/卸载
- 杀毒软件可能拦截，需添加信任
- 服务本身不显示界面，只负责启动主程序
- 卸载程序前请先卸载服务

## 故障排除

**服务安装失败**
- 检查是否授予了管理员权限
- 关闭杀毒软件后重试
- 查看日志：`%APPDATA%\QuickLauncher\error.log`

**服务已安装但程序未启动**
- 打开服务管理器检查服务状态
- 查看服务日志：事件查看器 → Windows 日志 → 应用程序

**如何切换回注册表方式**
1. 运行 `uninstall_service.bat` 卸载服务
2. 在程序设置中启用"开机自启"（注册表方式）
