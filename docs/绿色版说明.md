# 绿色版与安装版对比

## 安装版（当前实现）
- 安装时勾选"开机自启" → 自动安装服务
- 卸载时自动清理服务
- 适合：普通用户

## 绿色版（免安装）
- 首次运行：不启用服务
- 用户勾选"开机自启" → 弹出提示 → 请求管理员权限 → 安装服务
- 用户取消勾选 → 卸载服务
- 适合：便携使用、测试环境

## 技术实现

### 区别点
1. **安装版**：安装包自动处理服务
2. **绿色版**：运行时动态安装服务

### 服务路径问题
- 服务需要绝对路径
- 绿色版可能在任意位置运行
- 解决：服务记录当前 exe 路径

### 权限提示
绿色版勾选"开机自启"时显示：
```
开机自启需要安装系统服务（需要管理员权限）

是否继续？
- 是：弹出 UAC，安装服务
- 否：使用注册表方式（启动较慢）
```

## 当前代码已支持

`ui/config_window/settings_panel.py` 中的逻辑：
- 优先尝试服务
- 失败自动降级到注册表

**绿色版无需修改代码**，已经支持！
