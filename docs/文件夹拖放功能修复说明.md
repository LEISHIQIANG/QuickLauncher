# 文件夹拖放功能修复与增强说明

## 修复时间
2026-02-05

## 问题描述

1. **FileSystemEventHandler 未定义错误**
   - 当 watchdog 库未安装时，`core/folder_watcher.py` 中使用 `FileSystemEventHandler` 导致 `NameError`
   - 错误信息：`NameError: name 'FileSystemEventHandler' is not defined`

2. **提示弹窗未应用主题**
   - 拖入文件夹导入时的各种提示弹窗使用原生 `QMessageBox`
   - 未跟随应用主题（深色/浅色模式）

3. **功能缺失**
   - 只支持识别 `.lnk` 和 `.exe` 文件
   - 不支持识别拖入文件夹下的子文件夹

## 修复内容

### 1. 修复 `core/folder_watcher.py` - FileSystemEventHandler 错误

**文件位置：** `core/folder_watcher.py`

**修改内容：**
- 在 watchdog 库不可用时创建虚拟基类 `FileSystemEventHandler`
- 避免 NameError，确保代码在没有 watchdog 的环境下也能正常运行
- 更新事件处理器，监听文件夹的创建、删除、移动事件

```python
try:
    from watchdog.observers import Observer
    from watchdog.events import FileSystemEventHandler
    WATCHDOG_AVAILABLE = True
except ImportError:
    WATCHDOG_AVAILABLE = False
    logging.warning("watchdog库未安装,文件夹自动同步功能不可用")
    # 创建虚拟基类以避免 NameError
    class FileSystemEventHandler:
        """虚拟基类 - watchdog 不可用时使用"""
        pass
```

**事件处理增强：**
- `on_created`: 监听文件和文件夹的创建
- `on_deleted`: 监听文件和文件夹的删除
- `on_modified`: 监听文件的修改（文件夹修改不触发）
- `on_moved`: 监听文件和文件夹的移动/重命名

### 2. 更新 `core/folder_scanner.py` - 支持子文件夹识别

**文件位置：** `core/folder_scanner.py`

**修改内容：**
- 添加 `INCLUDE_FOLDERS = True` 配置项
- 实现 `_create_folder_shortcut()` 方法创建文件夹快捷方式
- 修改扫描逻辑，识别三类内容：
  1. **子文件夹** - 自动识别并添加到图标栏
  2. **.exe 文件** - 可执行程序
  3. **.lnk 文件** - 快捷方式链接

**新增方法：**
```python
@staticmethod
def _create_folder_shortcut(folder_path: Path) -> ShortcutItem:
    """从文件夹创建快捷方式对象"""
    shortcut = ShortcutItem()
    shortcut.id = str(uuid.uuid4())
    shortcut.name = folder_path.name
    shortcut.target_path = str(folder_path)
    shortcut.type = ShortcutType.FOLDER
    return shortcut
```

### 3. 创建 `ui/styles/themed_messagebox.py` - 主题化消息框

**文件位置：** `ui/styles/themed_messagebox.py`（新建）

**功能：**
提供完整的主题化消息框替代 `QMessageBox`，包括：

- **消息框类型：**
  - `Information` - 信息提示（ℹ️）
  - `Question` - 询问确认（❓）
  - `Warning` - 警告提示（⚠️）
  - `Critical` - 错误提示（❌）

- **按钮支持：**
  - `Ok` - 确定
  - `Cancel` - 取消
  - `Yes` - 是
  - `No` - 否

- **主题自适应：**
  - 自动检测父窗口的主题设置（深色/浅色）
  - 圆角窗口，毛玻璃效果
  - 统一的 Apple 风格设计

**使用示例：**
```python
# 信息提示
ThemedMessageBox.information(self, "标题", "内容文本")

# 询问确认
reply = ThemedMessageBox.question(self, "标题", "是否继续?")
if reply == ThemedMessageBox.Yes:
    # 用户点击"是"
    pass

# 警告提示
ThemedMessageBox.warning(self, "警告", "警告内容")

# 错误提示
ThemedMessageBox.critical(self, "错误", "错误内容")
```

## 注意事项

1. **字体文件**
   - 如果使用思源黑体，请确保 `resources/fonts/SourceHanSansSC-Regular.otf` 文件存在
   - 如字体文件缺失，会自动回退到系统字体

2. **性能考虑**
   - 文件夹监听使用非递归模式，只监听直接子项
   - 防抖动机制避免频繁触发同步（2秒间隔）
   - 大文件夹扫描可能需要几秒钟

3. **兼容性**
   - 已测试 PyQt5/PyQt6 兼容性
   - Windows 10/11 完全支持
   - macOS/Linux 可能需要额外测试
