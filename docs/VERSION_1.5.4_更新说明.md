# QuickLauncher V1.5.4 更新说明

## 📅 发布日期
2026-03-29

---

## ✨ 新增功能

### 1. 快捷方式支持管理员权限运行
- **编辑快捷方式**窗口新增"以管理员身份运行"选项
- 勾选后，通过中键弹窗启动的应用将自动以管理员权限运行
- 适用于需要管理员权限的应用（如注册表编辑器、系统工具等）
- 使用 Windows `runas` 动词实现，兼容性好

### 2. 中键弹窗行数自定义控制
- 新增 `popup_max_rows` 设置项（默认值：3）
- 可自定义中键弹窗每列显示的最大行数
- 支持动态调整弹窗布局，适应不同使用习惯

### 3. 免安装绿色版支持
- 支持免安装运行，解压即用
- 绿色版首次运行不启用服务
- 用户勾选"开机自启"时动态安装服务
- 适合便携使用、测试环境

---

## 🚀 性能优化

### 1. 开机自启动方式优化
- **从注册表启动改为任务计划程序启动**
- 启动时机提前 5-10 秒
- 任务计划程序在用户登录时立即触发，不等桌面加载
- 可设置延迟启动，避免开机卡顿
- 注册表方式已废弃，仅在禁用时清理历史残留

### 2. 打包体积优化
- 使用 `build_optimized.spec` 优化打包配置
- 排除不必要的 Qt 模块（Quick、QML、WebEngine、Multimedia 等）
- 过滤无用的二进制文件（OpenGL、D3D 编译器等）
- 启用 UPX 压缩和代码优化
- **安装版：23MB，免安装版（压缩包）：21MB**

### 3. 钩子稳定性优化
- 优化 `hooks.cpp` 实现
- 改进钩子注册和卸载流程
- 提升中键弹窗响应稳定性
- 减少内存占用

## 🔧 代码优化

### 1. 数据模型清理
- 移除已废弃的 `QuickSearchItem` 数据模型
- 清理 `command_delay` 等过时字段
- 优化数据序列化和反序列化逻辑
- 减少冗余代码

### 2. 自启动管理器重构
- 重构 `auto_start_manager.py`
- 优先使用任务计划程序
- 注册表方式作为兜底方案
- 新增 `task_scheduler_manager.py` 专门管理任务计划

### 3. 执行器改进
- `_shell_execute_open()` 支持 `runas` 动词
- `_execute_file()` 根据设置使用管理员权限启动
- 管理员模式下跳过窗口激活检查（避免权限冲突）

## ⚠️ 注意事项

### 升级建议
1. 建议先备份配置文件（`config/data.json`）
2. 卸载旧版本后再安装新版本
3. 首次启动会自动迁移配置

### 已知问题
1. 管理员权限运行的程序无法拖放文件（Windows 安全限制）
2. 绿色版首次启用开机自启需要 UAC 授权
3. 任务计划程序方式需要 Windows 7 及以上系统

### 兼容性
- Windows 7 / 8 / 10 / 11
- 支持 PyQt5 和 PyQt6
- 完全兼容 V1.5.3 配置文件
