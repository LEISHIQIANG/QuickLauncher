# 对话框居中优化说明

## 优化时间
2026-02-05

## 问题描述
拖入文件夹到左侧分类栏时，以及其他二级弹窗出现时，弹窗位置不正确，没有居中在主窗口（ConfigWindow）中心。

部分对话框使用 `self.parent().geometry()` 进行居中，但当对话框的 parent 不是主窗口而是子组件（如 FolderPanel）时，会导致居中在错误的位置。

## 解决方案

### 1. 创建统一的居中辅助函数

**文件：** `ui/utils/dialog_helper.py`（新建）

**功能：**
提供统一的对话框居中函数，确保所有对话框都居中在主窗口（ConfigWindow）中。

**核心函数：**
```python
def center_dialog_on_main_window(dialog: QWidget):
    """
    将对话框居中在主窗口（ConfigWindow）中

    使用 parent.window() 方法获取顶级窗口，
    即使 parent 是子组件也能找到 ConfigWindow
    """
    if not dialog:
        return

    parent = dialog.parent()
    if not parent:
        return

    # 找到顶级窗口（ConfigWindow）
    top_window = parent.window()
    if not top_window:
        return

    # 获取主窗口的几何信息
    main_geo = top_window.geometry()
    dialog_size = dialog.size()

    # 计算居中位置
    x = main_geo.x() + (main_geo.width() - dialog_size.width()) // 2
    y = main_geo.y() + (main_geo.height() - dialog_size.height()) // 2

    # 移动对话框到居中位置
    dialog.move(int(x), int(y))
```

**关键点：**
- 使用 `parent.window()` 而不是 `parent`
- `window()` 方法会返回顶级窗口，即使 parent 是嵌套的子组件
- 这确保所有对话框都居中在 ConfigWindow，而不是某个面板上

### 3. 修改示例

#### 修改前（错误的居中方式）
```python
def showEvent(self, event):
    super().showEvent(event)
    self.adjustSize()

    if self.parent():
        parent_rect = self.parent().geometry()  # ❌ 错误：可能是子组件
        self.move(
            parent_rect.x() + (parent_rect.width() - self.width()) // 2,
            parent_rect.y() + (parent_rect.height() - self.height()) // 2
        )
```

**问题：**
- 当 `self.parent()` 是 FolderPanel 时，居中在 FolderPanel 上
- 导致对话框出现在窗口左侧或其他错误位置

#### 修改后（正确的居中方式）
```python
from ui.utils.dialog_helper import center_dialog_on_main_window

def showEvent(self, event):
    super().showEvent(event)
    self.adjustSize()
    # 使用统一的居中函数，确保居中在主窗口
    center_dialog_on_main_window(self)
```

**优点：**
- 始终居中在 ConfigWindow（主窗口）
- 代码简洁统一
- 易于维护

## 窗口居中的技术细节

### Qt 窗口几何信息

```python
# 获取窗口几何信息
geometry = window.geometry()
geometry.x()      # 窗口左上角 X 坐标（屏幕坐标）
geometry.y()      # 窗口左上角 Y 坐标（屏幕坐标）
geometry.width()  # 窗口宽度
geometry.height() # 窗口高度
```

### 居中计算公式

```python
# 对话框居中位置计算
dialog_x = window_x + (window_width - dialog_width) // 2
dialog_y = window_y + (window_height - dialog_height) // 2
```

### parent() vs window()

| 方法 | 返回值 | 用途 |
|-----|--------|------|
| `widget.parent()` | 直接父组件 | 获取父子关系中的父组件 |
| `widget.window()` | 顶级窗口 | 获取包含该组件的顶级窗口 |

**示例：**
```python
# 假设层级：ConfigWindow -> FolderPanel -> dialog
dialog.parent()        # 返回 FolderPanel
dialog.parent().window()  # 返回 ConfigWindow ✅
```

## 代码审查要点

### 检查居中逻辑是否正确

❌ **错误模式：**
```python
# 直接使用 parent().geometry()
parent_rect = self.parent().geometry()
x = parent_rect.x() + ...
```

✅ **正确模式：**
```python
# 使用统一函数
from ui.utils.dialog_helper import center_dialog_on_main_window
center_dialog_on_main_window(self)
```

或

✅ **正确模式：**
```python
# 使用 window() 获取顶级窗口
top_window = self.parent().window()
main_geo = top_window.geometry()
x = main_geo.x() + ...
```
