# 配置文件迁移说明

## 变更内容

### 1. 配置路径变更
- **旧路径**: `%APPDATA%\QuickLauncher`
- **新路径**: `安装目录\config`

### 2. 自动迁移机制

#### 安装版
- 安装程序会在安装完成后自动检测并迁移旧配置
- 迁移完成后会删除旧配置目录

#### 免安装版
- 首次启动时自动检测并迁移
- 使用 `portable_launcher.py` 启动

### 3. 迁移内容
- `data.json` - 主配置文件
- 所有其他配置文件

### 4. 图标缓存
- 图标继续存储在 `安装目录\icons`

## 技术实现

### 核心模块
- `core/config_migrator.py` - 配置迁移工具
- `core/data_manager.py` - 使用新配置路径
- `scripts/installer.iss` - 安装时执行迁移

### 迁移逻辑
1. 检测旧配置是否存在
2. 检测新配置是否已存在
3. 如果需要迁移，移动所有文件到新位置
4. 删除旧配置目录

## 兼容性
- 支持从旧版本升级
- 支持全新安装
- 支持免安装版和安装版
