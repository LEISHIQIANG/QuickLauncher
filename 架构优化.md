#### 任务 1.2:DataManager 三段拆解(2 周)

**目标**:把 603 行上帝对象拆为 `AppPaths` / `PersistenceEngine` / `ConfigServices` 三个职责清晰的类。

**新结构**:

```python
class AppPaths:
    """只读路径聚合"""
    install_dir: Path
    app_dir: Path
    data_file: Path
    icon_repo_file: Path
    icons_dir: Path
    auto_backup_dir: Path
    history_dir: Path
    recovery_dir: Path

class PersistenceEngine:
    """加载/保存/恢复/事务日志"""
    def __init__(self, paths: AppPaths, history: ConfigHistory, ...): ...
    def load(self) -> AppData: ...
    def save(self, data: AppData, immediate: bool = False): ...
    def restore(self, snapshot_id: str): ...

class ConfigServices:
    """组合子服务容器"""
    def __init__(self, paths: AppPaths, engine: PersistenceEngine, ...):
        self.shortcuts = ShortcutService(engine)
        self.folders = FolderService(engine)
        self.icons = IconRepository(paths.icons_dir)
        ...

class DataManager:
    """Facade,保持旧 API 兼容"""
    def __init__(self):
        self.paths = AppPaths()
        self.engine = PersistenceEngine(self.paths)
        self.services = ConfigServices(self.paths, self.engine)
        self.data = self.engine.load()
```

4. 删除 22 个 `_get_*` 懒加载方法
5. 保留 `DataManager` 作为 facade,公开 API 不变
6. `core.set_data_manager()` 标记 deprecated,2 个版本后删除

**验收标准**:

#### 任务 1.3:ServiceRegistry 替换全局回调(1 周)

**目标**:把 `core/__init__.py` 的无类型 `_callbacks: dict` 升级为强类型 `ServiceRegistry`。

**新文件**:`bootstrap/service_registry.py`

```python
from typing import Literal, Callable, Any

CallbackName = Literal[
    "show_config_window", "quit_app", "restart_app",
    "show_log", "show_about", "show_help",
    "show_diagnostics", "show_shortcut_health",
    "show_config_history", "clean_icon_cache",
    "reload_hooks", "open_data_dir", "open_install_dir",
]


class ServiceRegistry:
    _instance = None
    _lock = threading.Lock()

    def __init__(self):
        self._services: dict[str, Any] = {}
        self._factories: dict[str, Callable] = {}

    def register(self, name: str, service: Any) -> None: ...
    def register_factory(self, name: str, factory: Callable) -> None: ...
    def resolve(self, name: str) -> Any: ...
    def resolve_callback(self, name: CallbackName) -> Callable: ...


# core/__init__.py 适配器
def register_callback(name: str, callback):
    """已废弃,改用 bootstrap.service_registry。"""
    warnings.warn(
        "core.register_callback 已废弃,请改用 bootstrap.service_registry",
        DeprecationWarning,
        stacklevel=2,
    )
    bootstrap.service_registry.resolve_callback(name)
```

4. 内部调用点 `core.call_callback("show_config_window")` 改为 `service_registry.resolve("show_config_window")()`

**验收标准**:

# bootstrap/env_setup.py
def apply_default_env_overrides() -> None:
    """应用默认环境变量,使用 setdefault 避免覆盖用户设置。"""
    overrides = {
        "PYTHONUTF8": "1",
        "PYTHONIOENCODING": "utf-8",
        "QT_AUTO_SCREEN_SCALE_FACTOR": "0",
        "QT_ENABLE_HIGHDPI_SCALING": "1",
        "QT_SCALE_FACTOR_ROUNDING_POLICY": "PassThrough",
        "QT_LOGGING_RULES": "qt.qpa.fonts.warning=false",
    }
    for key, value in overrides.items():
        if key not in os.environ:
            os.environ[key] = value


def sanitize_gui_env() -> None:
    """移除可能干扰 Qt/GUI 启动的 Python/Qt 环境变量。"""
    for k in list(os.environ.keys()):
        if k.upper().startswith("PYTHON"):
            os.environ.pop(k, None)
    for k in ("QT_PLUGIN_PATH", "QT_QPA_PLATFORM_PLUGIN_PATH",
              "QML2_IMPORT_PATH", "QML_IMPORT_PATH"):
        os.environ.pop(k, None)

**任务 2.1-2.3 详细说明**:

- **2.1 i18n 全量翻译**:逐条审查 `_EN_US` 字典 1098 条,统一为完整英文短句/标准 UI 术语;错误消息必须含操作建议。验证:英语环境下零中文 fallback。
- **2.2 TranslationProvider**:定义 `TranslationProvider` Protocol,`tr()` / `_repr_en()` 改为走 Provider,默认 `DictionaryProvider`,提供 `register_translation_provider(name, provider)` 接口。
- **2.3 broad-exception 治理**:分批治理,每批锁定 `--max-total` 阈值。替换规则:
  - OS 调用 → `except OSError`
  - JSON/YAML → `except (json.JSONDecodeError, yaml.YAMLError)`
  - 文件 I/O → `except (FileNotFoundError, PermissionError, OSError)`
  - 子进程 → `except (subprocess.SubprocessError, OSError)`
