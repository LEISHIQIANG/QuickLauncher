# 截图二维码识别

框选屏幕上的矩形区域，识别其中的单个二维码内容，并在命令结果面板中直接显示纯文本。结果面板提供复制动作；当内容是可访问链接时，也提供打开动作。

## 命令

- ID：`screenshot-qr`
- 别名：`screenshot_qr`、`qr`、`二维码识别`、`截图二维码`
- 输出：二维码原始内容，复制动作，可访问链接的打开动作

## 使用

在 QuickLauncher 中输入 `qr` 或 `二维码识别`，拖拽框选二维码区域，然后点击“识别”或按 Enter。右键、Esc 或“取消”会退出截图。

## 权限

- `builtin.command`：注册为可在内置命令下拉中选择的命令。
- `process.run`：启动宿主管理的常驻 worker，提前加载 Qt 和二维码解码器。
- `clipboard.write`：结果面板提供复制内容动作。
- `open.url`：二维码内容是 URL 时，结果面板提供打开链接动作。

## 运行库

插件只内置 `zxingcpp` 的 Windows x64 Python 3.12/3.13 二进制解码模块，不包含完整 OpenCV、Pillow 或 PyQt5。截图窗口使用 QuickLauncher 本体已有的 PyQt5，图像读取使用本体已有的 Pillow。运行时会在启动空闲期常驻预热，执行命令时直接复用。

## 故障排查

- 未识别到二维码：请尽量只框选二维码本体，并保留少量白边。
- 解码组件不可用：目标运行环境的 Python 版本可能不是 3.12/3.13，需补充对应版本的 `zxingcpp` 二进制模块。
- 截图窗口未出现：确认 QuickLauncher 本体包含 PyQt5，且未被安全软件拦截 helper 子进程。
